# Hardware Overview

This section contains guides for hardware setup and configuration. We will be using the HOBO data logging ecosystem by OnSet throughout site visits. Before arrving at any facility, you will need to download HOBOconnect on your mobile device or iPad.

For more details, navigate to the manufacturers site: 
https://www.onsetcomp.com/products/software/hoboconnect

## Module Objectives

- Prepare to deploy sensors and data loggers at site visits
- Be able to distinguish situations to use a data logger
- Setup and configure analog sensors and data loggers
