# MX1104 / MX1105


## Setup

Start by openning the HOBOconnect app on your mobile device and navigate to the devices tab at the bottom of the screen.

Ensure the data logger is on by tapping the buttons on its front face. 
On the back, find the serial number under the ONSET MX1104 label. 


Select the device with the matching serial number. A connection should be established and your mobile device should greet you with a new menu.

![menu](../assets/hardware-stuff/Connected_Menu.png#only-light)
![menu](../assets/hardware-stuff/Connected_Menu-Dark.png#only-dark)


!!! success

    Press the "beep logger" button to confirm a successful connection.

Inside the Customize Configuration & start menu, set the following settings: 

#### Logger Name & Group

Locate the five-digit ITAC identification number under the barcode on the back of the data logger, and assign this value to the logger's name. Set the logger's group to none.

| Setting | Value |
|---|---|
| Name | ITAC ID |
| Group | None |
