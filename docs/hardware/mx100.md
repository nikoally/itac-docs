# MX100 - Temperature Logger

We need to know the temperature of the air in the compressor room. We will be using the HOBOconnect app to configure a temperature logger.

## Setup

Start by opening the HOBOconnect app on your mobile device and navigate to the devices tab at the bottom of the screen.

On the side of the logger, find the serial number above the ONSET MX100 label.

Ensure the data logger is on by tapping the button on its front face. The device should appear in the app.

Select the device with the matching serial number. A connection should be established and your mobile device should greet you with a new menu.

!!! success

    Press the *illuminate logger* button to confirm a successful connection. Lights on the front of the logger should briefly illuminate.

Once a successful connection has been confirmed, tap the *Customize Configuration & Start* menu to set up the temperature logger for use. Using the guide below, set up the logger.

#### Logger name & group

| Setting | Value |
|---|---|
| Name | *ITAC I.D.* |
| Group | None |

The ITAC I.D. is a five-digit number on the back of the logger.

#### Logger settings

| Setting | Value |
|---|---|
| Logging interval | 10 minutes |
| Start logging | On save |
| Stop logging | Stop when memory fills |
| Show LED | On |
| Bluetooth always on | On |
| Capture location data | Off |
| Update saved configuration | On |

#### Temperature

| Setting | Value |
|---|---|
| Sensor / channel name | *(blank)* |

#### Alarms

| Setting | Value |
|---|---|
| Low | Off |
| High | Off |

#### LI-COR Cloud Connection

| Setting | Value |
|---|---|
| LI-COR Cloud Connection | Off |

## Deployment

After updating the logger settings, press the *Start* button to start logging and return to the main menu.

In the menu, press *View Live Data*. Confirm readings appear and the temperature is reasonable (roughly matches the room).

Photograph the ITAC I.D. now, before mounting. It is on the back of the logger and will be hidden once mounted.

This logger is magnetic; mount on a wall or strut at ~5ft, away from hot equipment, direct airflow from fans, and doorways.

Photograph the mounted logger and the surroundings of its new home. Include a recognizable landmark (door, equipment tag, sign) visible across the photos so someone else can find the logger.

### Checkout

!!! warning

    Make sure to check out all data loggers in Airtable before leaving the site. We need to be able to tell the customer where to find them.

In the ITAC Airtable, navigate to the *Assessment Data* tab and locate the facility that the logger has been left at. Locate the *Data Logger* row and press the *New Checkout* button. Select all relevant ITAC I.D.s from the *Asset* dropdown menu. Provide a description of the location in the room where the loggers are placed. Upload all relevant images of the logger and the surrounding area. Press *Create* to submit.

## Data Extraction

After some time at the facility, the data logger will be returned to ITAC for processing. To extract the data from the logger, reconnect the device to the *HOBOconnect* app. In the main menu, press *Download Data*, then *Export and Share*, and save the .xlsx file to the *sensor data* folder inside the correct *current assessments* folder in the ITAC OneDrive.

### Returning logger

After extracting the data, the data logger must be marked as returned in Airtable before being returned to the Packout (the equipment staging area). To do this, navigate to the *Assessment Data* tab and locate the facility that the logger was left at. Locate the *Data Logger* row and press the *New Return* button. Select all relevant ITAC I.D.s from the *Asset* dropdown menu. If the logger seems damaged, add a description to the notes section. Press *Create* to submit.
